Bienvenido al Visualizador de Sensores M5StackFire

¿Qué es esto?

Esto es una web para visualizar datos en tiempo real provenientes de un sensor M5StackFire. Los datos de aceleración y otros parámetros son enviados desde el dispositivo y visualizados en tiempo real.

¿Cómo funciona?

El M5StackFire manda sus datos a través de una conexión al servidor, y usamos una interfaz web para mostrar los valores de aceleración y otros parámetros en gráficos y coordenadas.
